PROJECT DESCRIPTION:

 

The Relationship Opening Tool (ROT) provides support with opening client relationships and products. When preparing forms, ROT also searches for the correct contract forms - in a way that accommodates individual client needs and particular product requirements.

The dialog leads the user through the structured input fields which must be answered correctly and in full. The completed data is integrated automatically in the relevant forms and prepared in a contract package that meets CDB requirements.

ROT allows the user to open CIFs and products, and to create, replace, modify or delete simultaneously any documents that are needed for a client relationship. On the basis of the CIF basic data and the selected products, the system generates different questions which must be answered in full. This allows the user to print out directive-compliant contracts (including checklist), and at the same time create CIFs and other products. The current release offers the following functions:

 

§  Basic information, CIF data, Sending correspondence, Identification, Segment-specific and non-segment-specific product selection and powers of attorney, Shopping basket, Beneficial ownership, US withholding tax, EU taxation of savings income, KYC, Varia, Print
